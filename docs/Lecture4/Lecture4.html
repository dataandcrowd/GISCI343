Melt and Cast

  • Pandas melt() function is used to change the DataFrame format from wide to long.
  • It‚Äôs used to create a specific format of the DataFrame object where one or more columns work as identifiers.
  • All the remaining columns are treated as values and unpivoted to the row axis and only two columns ‚Äì variable and value.
  • We can use pivot() function to unmelt the output DataFrame back to the original data frame.

What is GIS

  • Graphical Information Systems:
  • Computer system for capturing, storing, checking, and displaying data related to positions on Earth‚Äôs surface

What is GIS

  • All data in GIS are ‚Äògeoreferenced‚Äô meaning it has both;
    • Attribute (what it is)
    • Location (where it is, a known location)

Reference vs Thematic Maps

  • Reference Maps: used to communicate location on more static data points
    • To ‚Äòpin point‚Äô data on a map
    • Descriptive
  • Thematic Maps: used to highlight a spatial relationship
    • To ‚Äòstudy a theme‚Äô within a map
    • Explanatory

‚ÄúThe visualisation of road networks to improve road safety measures‚Äù This is typically a Thematic Map ‚Äî specifically a transportation or road safety map. These maps focus on a specific theme or purpose (in this case, road safety), and visualize relevant features like crash hotspots, speed limits, traffic volumes, or intersections.

‚ÄúThe visualisation of the earth‚Äôs surface, showing its elevation‚Äù This describes a Topographic Map, which represents elevation through contour lines, shading, or colour gradients to depict terrain and landforms.

‚ÄúNavigation tools such as Google Maps or CityMapper‚Äù These fall under Reference Maps or Navigational Maps, designed for general location-finding, route planning, and wayfinding. While they use various thematic layers (e.g., traffic, transit), their primary function is spatial orientation.

What is Geospatial Data?

  • Geospatial data (or spatial data in particular areas in the world) is a data frame that contains information about a specific location, which can be analysed to better understand that location. GIS enables this spatial data to be processed and analysed.
  • Two types: Vector and Raster

Vector and Raster

  • Vector: points, lines and polygons

  • Raster: Imagery or satellite data that are formed from a grid of pixels.

GIS Data Management

  • GeoTiff files are commonly used for raster data
  • Shapefile files are commonly used for vector data. Here‚Äôs a file extension‚Ä¶

Which data to use

  • Vector datasets and methods dominate the social sciences because human settlements and processes tend to have discrete borders (e.g., transport)
  • Raster datasets and methods dominate many environmental sciences because of the reliance on remote sensing data

Don‚Äôt forget about me!

  • GeoJSON (.geojson / .json) ‚Äì Web-friendly, lightweight format for encoding geographic data structures.
  • KML/KMZ ‚Äì Used primarily in Google Earth; KMZ is a zipped KML.
  • GPKG (GeoPackage) ‚Äì A modern SQLite-based format supporting both vector and raster data.
  • File Geodatabase (GDB) ‚Äì ESRI‚Äôs proprietary format allowing complex datasets and relationships.
  • CSV + coordinates ‚Äì Plain text tabular data that can be georeferenced using lat/lon columns.
  • DXF/DWG ‚Äì CAD files that can be converted to GIS-compatible formats.
  • GPX (GPS Exchange Format) ‚Äì Typically used for GPS data (routes, tracks, waypoints).
  • MapInfo TAB / MIF ‚Äì Formats used in MapInfo software

Modifiable Arial Unit Problem (MAUP)

MAUP refers to the cartographic representation of data whose attributes are significantly influenced by the spatial scale used

Two key Aspects:

  • Scale Effect: Changing the size of the spatial units (e.g., from neighbourhoods to districts) can alter statistical results, such as means or totals.
  • Zoning Effect: Altering the shape or configuration of spatial units, even if the scale remains constant, can also impact results.

Spatial Join sjoin

  • merging attributes from two geometry layers based on their spatial relationship

  • Different parts of this operations:

    • The GeoDataFrame to which we want add information
    • The GeoDataFrame that contains the information we want to add
    • The spatial relationship we want to use to match both datasets (intersects, contains, within)
    • The type of join: left or inner join

Cities and countries

In this case, we want to join the cities dataframe, containing Point geometries, with the information of the countries dataframe, containing Polygon geometries.

To match cities with countries, we‚Äôll use the within spatial relationship.

joined = gpd.sjoin(

Summary: üåê Core GIS Concepts

  • GIS = Attribute + Location ‚Äî everything is georeferenced
  • Vector vs Raster:
    • Vector: points, lines, polygons
    • Raster: grids of pixels (e.g., satellite images)
  • Reference vs Thematic Maps:
    • Reference = location/navigation
    • Thematic = patterns/relationships
  • Projections & CRS:
    • Map distortion comes from projecting 3D ‚Üí 2D
    • Use EPSG codes (e.g., 4326 for WGS84)
  • MAUP:
    • Scale Effect: unit size changes results
    • Zoning Effect: how units are drawn matters
  • Spatial relationships:
    • within, intersects, contains, touches, etc.
